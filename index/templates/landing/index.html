Features

Some features of EK's django boilerplate.

Authentication

This project, has API for authentication based on DRF Token and JWT. there is authentication views for MTV approach too. checkout authentication app in order to see and change these features.

Tested

Every API in this project: Authentication, User, Profile, OTP, OTL etc. are tested and you can find them in each apps directory under tests.

OTP/OTL

OTP(One time password) and OTL (One time links) are implemented in this project with redis. there is also a redis helpers file which will help you to store and retrieve data from redis.

Dockerized

This project is fully dockerized and there is docker-compose file for Every situation. you can use the dockerized compose files in test,producion,local and debugging.

Mailpit

Do you want to see the emails you send in your project in a place more like a real mailing service ? you have mailpit in order to see the emails intead of using console backend which django provides.

Tailwindcss

pre configured tailwindcss in this project. you are free to create and style your pages in a way you want not the way you are told.
